Once all files are set up please compile in the following order using command: javac <filename>.java
  Messages.java 
  Type.java
  Parser.java
  Node.java
  Handler.java
  TCPClient.java
  TCPServer.java
  BFSTree.java
  LeaderElection.java
  Main.java
Please note that Main does not have a natural exit and requires cntrl+c to finish
